# 🧠 AI/ML Internship Projects – DevelopersHub Corporation

This repository contains a collection of machine learning and AI projects completed as part of the AI/ML Engineering Internship at DevelopersHub Corporation.

The projects focus on real-world datasets and practical machine learning applications including data analysis, regression, classification, and large language model-based chatbots.

---

## 📌 Selected Tasks

This repository includes the following completed tasks:

---

## 📊 Task 1: Exploratory Data Analysis – Iris Dataset

### 🎯 Objective
To explore and visualize dataset patterns using statistical analysis and data visualization techniques.

### 📂 Dataset
Iris Flower Dataset (Seaborn / CSV format)

### 🛠️ Techniques Used
- Pandas for data loading and inspection
- Statistical summary using `.describe()` and `.info()`
- Data visualization using Matplotlib and Seaborn
  - Scatter plots
  - Histograms
  - Box plots

### 📈 Outcome
Gained understanding of feature relationships, distributions, and data patterns.

---

## ❤️ Task 3: Heart Disease Prediction

### 🎯 Objective
To build a machine learning model that predicts the risk of heart disease based on medical attributes.

### 📂 Dataset
Heart Disease UCI Dataset (Kaggle)

### 🛠️ Techniques Used
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Classification models:
  - Logistic Regression
  - Decision Tree
- Model evaluation:
  - Accuracy Score
  - Confusion Matrix
  - ROC Curve
- Feature importance analysis

### 📈 Outcome
Developed a binary classification model capable of predicting heart disease risk with evaluated performance metrics.

## 🤖 Task 4: General Health Query Chatbot (Prompt Engineering)

### 🎯 Objective
To develop a chatbot that responds to general health-related queries using a Large Language Model (LLM).

### 🛠️ Tools Used
- OpenAI GPT / Hugging Face Transformers
- Python API integration
- Prompt engineering techniques

### 💡 Key Features
- Conversational health assistant
- Safe response filtering
- Friendly and simple explanations
- Prompt-controlled behavior for medical safety

### 📈 Outcome
Built a functional AI chatbot capable of handling natural language health queries responsibly.

## 🏠 Task 6: House Price Prediction

### 🎯 Objective
To predict house prices using regression-based machine learning models.

### 📂 Dataset
House Price Prediction Dataset (Kaggle)

### 🛠️ Techniques Used
- Feature preprocessing (handling missing values, encoding)
- Feature scaling
- Regression models:
  - Linear Regression
  - Gradient Boosting Regressor
- Evaluation metrics:
  - Mean Absolute Error (MAE)
  - Root Mean Squared Error (RMSE)

### 📈 Outcome
Built a regression model capable of predicting property prices based on key features.
